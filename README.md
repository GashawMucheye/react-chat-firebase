# React realTime Chat Firebase

1. install firebase.

```js
npm i firebase
```

2. Firebase Configuration.

3. install chakra ui.

```js
npm i @chakra-ui/react @emotion/react @emotion/styled framer-motion

```

4. create components.

5. install react-router-dom

```js
npm i react-router-dom
```
